## ⑤工夫した点・こだわった点

- 地図で場所を指定できる
- 指定した場所の天気をOpenWeatherMapから取得
- 取得した天気に基づき、適した映像をPexelsから、音楽をSpotifyから取得

## ⑥難しかった点・次回トライしたいこと(又は機能)

- いつも配色にはこだわっているのですが、Spotifyのミュージックプレイヤーでは色の指定ができず、少し使いづらいと感じています。
